# NAME

get_cmd_line - return a pointer to the command line

# SYNOPSIS

char \*\
get_cmd_line (argc, argv)\
int argc;\
char \*\*argv;

# DESCRIPTION

*get_cmd_line* collects all the command line arguments together into a
single string. The pointer to the string is returned.

# DIAGNOSTICS

None.
