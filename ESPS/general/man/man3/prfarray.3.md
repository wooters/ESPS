# NAME

    pr_farray - print a float array

# SYNOPSIS

     
    void 
    pr_farray(arr, size, title)
        float  *arr;
        long    size;
        char    *title;

# DESCRIPTION

*pr_farray* prints *size* floating point values starting at location
*arr*. The numbers are formatted with 5 values per line. Before the
first line, *title* and *size* are printed.
