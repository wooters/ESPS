# NAME

record - see srecord, s16record, sgrecord, hprecord, decrecord, lrecord

# DESCRIPTION

Currently ESPS supports these systems for audio input. See the man page
for each record program for details.

*srecord* - mono record to disk or pipe from SPARCstation 8 bit codec
input.

*s16record* - mono or stereo record to disk or pipe for 16 bit Sparc
dbri A/D converter.

*sgrecord* - mono or stereo record to disk or pipe for SGI Indigo A/D
converter.

*hprecord* - mono or stereo record to disk or pipe for HP 700/C100.

*decrecord* - mono or stereo record to disk or pipe for DEC Alpha audio.

*lrecord* - mono or stereo record to disk or pipe for linux audio.
