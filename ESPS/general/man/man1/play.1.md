# NAME

play - see splay, s16play, sgplay, hpplay, decplay, lplay

# DESCRIPTION

Currently ESPS supports these systems for audio output. See the man page
for each play program for details.

*splay* - send ESPS sampled data to the SPARCstation 8 bit codec output.

*s16play* - send sampled data to Sun Sparc 16 bit dbri audio port.

*sgplay* - send sampled data to Indigo D/A converter.

*hpplay* - send sampled data to HP 700/C100 audio.

*decplay* - send sampled data to DEC Alpha audio.

*lplay* - send sampled data to linux audio.

# SEE ALSO

(1-ESPS).
