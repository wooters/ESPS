# NAME

eversion - get information about the current ESPS version

# SYNOPSIS

**eversion** \[ **-c** \]

# DESCRIPTION

*Eversion* provides information about the installed versions of ESPS and
*waves*+.

If no options are given when calling *eversion* it prints on standard
output the release number.

# OPTIONS

The following option is supported:

**-c**  
This option causes *eversion* to display the customer number.

# ESPS HEADERS

This program ignores all header items.

# ESPS PARAMETERS

This program does not access the parameter file.

# BUGS

None known.
