# NAME

efree - free an ESPS license

# SYNOPSIS

**efree**

# DESCRIPTION

*Efree*

frees an ESPS license that was checked out by *echeckout*. The license
can only be freed with this program by the same user that checked it
out.

If it is necessary to free a license that was checked out by another
user, the super-user will have to find the process id of the *echeckout*
and kill that process.

# ESPS HEADERS

This program ignores all header items.

# ESPS PARAMETERS

This program does not access the parameter file.

# OPTIONS

None.

# SEE ALSO

*echeckout*(1-ESPS), *elmd*(1-ESPS), *elmadmin*(1-ESPS)
