# NAME

emakecats - make *cat* versions of ESPS manual pages

# SYNOPSIS

**emakecats**

# DESCRIPTION

*Emakecats* makes the pre-stored nroff versions of the ESPS manual
pages. It makes them for sections 1, 3, and 5. This make the manual
pages come up fast when *eman* (1-ESPS) is used.

# SEE ALSO

*eman* (1-ESPS), *makewhatis* (1-ESPS)

    man1/*.1, 
    man1/*.1t, 
    man3/*.3, 
    man3/*.3t, 
    man5/*.5t
