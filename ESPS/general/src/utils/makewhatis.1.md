# NAME

makewhatis - makes the whatis file for eman -k

# SYNOPSIS

**makewhatis**

# DESCRIPTION

*Makewhatis* makes the *whatis* database file so that the **-k** option
of *eman* (1-ESPS) works.

# SEE ALSO

*eman* (1-ESPS), *emakecats* (1-ESPS)
