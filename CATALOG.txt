=== KEY FUNCTIONAL GROUPINGS ===

1. SPECTRAL ANALYSIS (15+ programs)
   - FFT, spectrograms, cepstrum, mel-scale, bark-scale, entropy-based

2. LPC / VOCAL TRACT ANALYSIS (10+ programs)
   - LPC coefficients, reflection coefficients, formants, synthesis

3. AUDIO I/O & CONVERSION (12+ programs)
   - Multiple platform support (Sun, HP, SGI, CONVEX, MASSCOMP, Linux, Mac)
   - Format conversions (SD, FEA_SD, FEA_SPEC, SPEC, SPHERE)

4. VECTOR QUANTIZATION (10+ programs)
   - Codebook design and management
   - Encoding and classification

5. INTERACTIVE VISUALIZATION (10+ programs)
   - ESPS plotting tools
   - X Windows integration via xwaves
   - Spectrograms, waveforms, 3D plots

6. PARAMETER/FEATURE MANIPULATION (20+ programs)
   - FEA file operations
   - Format conversions
   - Statistical analysis

7. SIGNAL PROCESSING FILTERS (10+ programs)
   - Digital filters (FIR, IIR, FFT-based)
   - Specialized filters (notch, windowed, codebook-based)

8. TESTING & COMPARISON (5+ programs)
   - Listening tests (A-B comparison)
   - Dynamic time warping
   - Data comparison

All programs support ESPS parameter files, ESPS Common, and extensive command-line options.
Most can read from/write to stdin/stdout for piping workflows.
